# Poker

## 專案簡介

玩家藉由輸入押注金額，按下押注後進行發牌，接著選擇是否換牌，最後由系統判斷牌型，並依照不同牌型的賠率計算中獎金額與更新總資金。

---

## 遊戲流程

1. 玩家輸入押注金額。
2. 按下「押注」按鈕。
3. 系統檢查押注金額是否合法。
4. 押注成功後，總資金會先扣除押注金額。
5. 玩家按下「發牌」按鈕。
6. 系統洗牌並發給玩家五張牌。
7. 玩家可以點選想要更換的牌，被選取的牌會翻成背面。
8. 按下「換牌」後，系統會替換被選取的牌。
9. 按下「判斷牌型」後，系統會判斷牌型並計算獎金。
10. 系統更新總資金，玩家可以開始下一局。

---

## 牌型與賠率

| 牌型 | 賠率 |
|---|---:|
| 皇家同花順 | 250 |
| 同花順 | 50 |
| 四條 | 25 |
| 葫蘆 | 9 |
| 同花 | 6 |
| 順子 | 4 |
| 三條 | 3 |
| 兩對 | 2 |
| 一對 | 1 |
| 雜牌 | 0 |
